# Collections Project Documentation

This directory contains documentation for the `collections` project, a real-time cryptocurrency trade screener and analysis tool.

## Overview

The `collections` project, also known as `SpreadAggregator`, is a .NET application designed to analyze and visualize real-time price deviations between multiple cryptocurrency exchanges.

## Architecture

For a detailed technical explanation of the project's architecture, data flow, and frontend implementation, please see the primary architecture document:

-   **[ARCHITECTURE.md](./ARCHITECTURE.md)**

This document provides comprehensive information on the system's design and is the recommended starting point for understanding the project.

## Other Documents

Other markdown files in this directory and its subdirectories (`proposals`, `sprints`) represent historical planning, task lists, and bug analyses. They can be consulted for historical context but may not reflect the current state of the implementation.
